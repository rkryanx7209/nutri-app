# 📱 Nutri-GT App

Versão mobile do sistema de agendamento e avaliação nutricional Nutri-GT, desenvolvido com Capacitor e Android Studio.

## 🔗 Versão web
[nutri.gt.tc](https://nutri.gt.tc/)

## 🛠️ Tecnologias usadas
- HTML
- CSS
- PHP
- Capacitor
- Android Studio

## 📌 Funcionalidades
- Login e acesso
- Agendamento de consultas
- Avaliações nutricionais
- Cadastro de pacientes
- Sincronizado com o site Nutri-GT

- ✅ Versão web online
- 🔧 Versão mobile em desenvolvimento local
